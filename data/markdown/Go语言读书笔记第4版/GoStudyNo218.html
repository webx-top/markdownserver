4. Channel
Channel 是 Go 实现 CSP 模型的关键，󶭟励󲙢通讯来实现共享。
在具体实现上，类似 FIFO 队列，多个 G 排队等待收发操作。同步模式，从排队链表中获
取个能与之交换数据的对象；异步模式，围绕数据缓冲区空位排队。
4.1 初始化
先了解个基本的数据类型。
SudoG 对 G 进󴘲包装，󳣐 WaitQ 则是 SudoG 排队链表。
runtime.h
struct SudoG
{
G* g;
uint32* selectdone;
SudoG* next;
SudoG* prev;
void* elem; // 发送或接收的数据。
int64 releasetime;
int32 nrelease; // -1 for acquire
SudoG* waitlink; // G.waiting list
};
chan.h
struct WaitQ
{
SudoG* first;
SudoG* last;
};
