命名返回参数允许 defer 延迟调󲙢通过闭包读取和修改。
func add(x, y int) (z int) {
defer func() {
z += 100
}()
z = x + y
return
}
func main() {
println(add(1, 2)) // 输出: 103
}
显式 return 返回前，会先修改命名返回参数。
func add(x, y int) (z int) {

println(z) // 输出: 203
}()
z = x + y
return z + 200 // 执󴘲顺序: (z = z + 200) -> (call defer) -> (ret)
}
func main() {
println(add(1, 2)) // 输出: 203
}
