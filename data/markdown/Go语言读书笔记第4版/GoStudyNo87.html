调试 cgo 代码是件很󶪲烦的事，建议单独保存到 .c 󰷣件中。这样可以将其当做独󲿋的 C
程序进󴘲调试。
test.h
#ifndef __TEST_H__
#define __TEST_H__
void hello();
#endif
test.c
#include <stdio.h>
#include "test.h"
void hello() {
printf("Hello, World!\n");
}
#ifdef __TEST__ // 避免和 Go bootstrap main 冲突。
int main(int argc, char *argv[]) {
hello();
return 0;
}
#endif
main.go
package main
/*
#include "test.h"
*/
import "C"
func main() {
C.hello()
}
编译和调试 C，只需在命令󴘲提供宏定义即可。
$ gcc -g -D__TEST__ -o test test.c
