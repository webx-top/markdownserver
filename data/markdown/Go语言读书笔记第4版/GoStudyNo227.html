gp := sg.g
sg.elem = nil
gp.param = nil
goready(gp)
}
}
总结规则如下：
发送：
•
向 nil channel 发送数据，阻塞。
•
向 closed channel 发送数据，出错。
•
同步发送: 如有接收者，交换数据。否则排队、阻塞。
•
异步发送: 如有空槽，拷󴻪数据到缓冲区。否则排队、阻塞。
接收：
•
从 nil channel 接收数据，阻塞。
•
从 closed channel 接收数据，返回已有数据或零值。
•
同步接收: 如有发送者，交换数据。否则排队、阻塞。
•
异步接收: 如有缓冲项，拷󴻪数据。否则排队、阻塞。
